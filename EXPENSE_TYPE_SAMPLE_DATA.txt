========================================
INSTRUCTIONS:
========================================

1. Go to: Master / Expense type creation
2. Click: + Create button
3. Enter data:
   - Expense Type: (Type the name from above)
   - Description: (Type the description from above)
   - Active: Check the checkbox (default checked)
   - Document Upload: (Skip or upload if needed)
4. Click: Submit
5. Repeat for each expense type

========================================
NOTE:
========================================

- All fields except "Expense Type" are optional
- Description helps users understand the expense category
- Active status should be checked for all active expense types
- Document upload is optional - you can add it later via Edit
